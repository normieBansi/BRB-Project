# Diagnostics Artifacts

This folder stores environment snapshots and troubleshooting captures used during lab validation.

## Files

- `Debian Server Config.txt`: Debian VM and host details.
- `Latest architecture.txt`: Current topology and VM resource plan.
- `Opnsense Config.txt`: OPNsense settings snapshot.
- `Opnsense.txt`: OPNsense shell command outputs and runtime diagnostics.
- `services.md`: Enabled services, IDS/logging configuration, and integrations.
